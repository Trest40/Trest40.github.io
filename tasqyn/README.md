# Tasqyn — портфолио-сайт проекта прогноза паводков

Автономный статический сайт-презентация проекта **Tasqyn / FloodCast** —
ML-модели прогноза месячной вероятности паводка для снеготаянных рек
Казахстана (река Ишим / Есиль и соседние бассейны). Без сборки и сервера.

## Как открыть

```bash
xdg-open index.html        # Linux  (или двойной клик)
```

Интернет нужен только для веб-шрифтов; данные, карта, гидрограф, графики
результатов и фоновое видео — локальные.

## Разделы

- **Hero** с фоновым видео настоящего наводнения (`assets/flood-bg.mp4`)
- **Метрики** — честная валидация: ROC-AUC 0.91, 10/14 паводков, 1/70 ложных тревог
- **Карта станций** — 6 бассейнов по реальным координатам, цвет = состояние уровня,
  клик → панель с показаниями и сезонной нормой
- **Гидрограф** — реальный ряд уровня Ишима у Петропавла, 1995–2022 (329 мес.)
- **Модель** — XGBoost, walk-forward валидация, отвергнутый наивный 1.00,
  переоткрытый механизм снеготаяния (`snow_cover_lag4` = 0.53, анимированные бары)
- **Результаты** — интерактивная галерея реальных графиков из `docs/figures/`
  (вкладки + увеличение по клику / лайтбокс)
- **Приложение Tasqyn** — макеты экранов Карта / Прогноз / История (Expo / RN)
- **Наводнения** и **Инженерия** — события 2024 г., стек, источники данных

## Данные

`data.js` (`window.FLOOD_DATA`) сгенерирован из реальных данных проекта:
`data/raw/river_level_monthly_aggregated.csv`, `data/processed/01_master.csv`,
`src/floodcast/basins.py`. Графики результатов — копии из `docs/figures/`.

## Структура

```
site/
├── index.html     # вся разметка
├── styles.css     # дизайн-система (светлый научный data-portal)
├── app.js         # карта, гидрограф, галерея, лайтбокс, бары
├── data.js        # реальные данные наблюдений
└── assets/        # иконка, flood-bg.mp4, графики результатов (.png)
```

## Дизайн

Светлый научный data-portal: navy + водяной синий, статусные цвета, шрифты
Spectral (заголовки) · IBM Plex Sans (текст) · IBM Plex Mono (данные). Тёмный
hero с видео наводнения. Без смайликов и AI-slop-маркеров.
